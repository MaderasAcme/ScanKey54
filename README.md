# 🔑 ScanKey54

Sistema de identificación de llaves con IA.

- 📸 Escaneo de imágenes en tiempo real
- 🤖 Detección por modelo ONNX
- 🧠 OCR y clasificación de marcas
- 🚀 Integración con Google Cloud y Expo

## 🛠️ Tecnologías
- React Native (Expo)
- FastAPI + Python
- ONNX Runtime
- Firebase Storage
